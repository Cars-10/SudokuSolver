Create Algorithms/CP/Dart/ directory structure. Port CP from C to Dart:

**cp.dart structure**:
- Use `List<List<int>> grid = List.generate(9, (_) => List.filled(9, 0))` for puzzle
- Use `List<List<int>> candidates` for bitsets (int type, bits 1-9 represent digits)
- Implement `propagate(int row, int col, int digit)` to eliminate candidates
- Implement `bool solve()` with backtracking recursion
- Use global `int iterations = 0` counter, increment BEFORE validity check
- Parse .matrix file from command line args (dart:io File)
- Initialize candidates: for empty cells, set to 0x3FE (bits 1-9), for filled cells set to 0

**Bitwise operations**:
- Check if digit available: `(candidates[row][col] & (1 << digit)) != 0`
- Remove candidate: `candidates[row][col] &= ~(1 << digit)`
- Add candidate back: `candidates[row][col] |= (1 << digit)`
- Count candidates: use bitwise AND to test each bit position

**Critical requirements**:
- Dart's bitwise operators: & (AND), | (OR), ~ (NOT), << (left shift)
- Must propagate constraints to row, column, and 3x3 box
- Backtracking must restore candidate sets correctly
- Print output format: puzzle, then "Solved in Iterations=67"
- Use integer division operator `~/` for box calculation: `int boxRow = (row ~/ 3) * 3;`
- **Avoid**: Don't use Set<int> for candidates (bitset is faster and matches reference). Don't use dynamic types (use explicit int).
