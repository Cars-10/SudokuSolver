- Pattern to find: `# Languages/{Language}/runMe.sh`
- Replace with: `# Algorithms/BruteForce/{Language}/runMe.sh`

Implementation:
```bash
# For each BruteForce language directory
for dir in Algorithms/BruteForce/*/; do
    script="$dir/runMe.sh"
    if [ -f "$script" ]; then
        # Extract language name from directory
        lang=$(basename "$dir")
        # Replace Languages/{lang} with Algorithms/BruteForce/{lang}
        sed -i '' "s|# Languages/$lang/runMe.sh|# Algorithms/BruteForce/$lang/runMe.sh|" "$script"
    fi
done
```

**What to avoid and WHY:**
- Don't use global find with sed -i without backup - use -i '' for BSD sed (macOS) or -i for GNU sed
- Don't modify DLX or CP scripts - they already have correct paths
- Don't change any functional code - only the header comment line
