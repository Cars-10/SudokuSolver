  <files>audit_tier1.sh</files>
  <action>
Create audit_tier1.sh at project root that:
1. Reads Tier 1 language list (can be hardcoded array from TIER1_LANGUAGES.md)
2. For each language:
   - Run: cd Languages/{LANG} && ./runMe.sh ../../Matrices/1.matrix
   - Parse stdout for "Solved in Iterations=NNN"

- Timeout after 60 seconds per language (Matrix 1 is fast)
- Use consistent output format for easy parsing
