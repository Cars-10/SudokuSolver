Create Algorithms/CP/Swift/ directory structure. Port CP from C to Swift:

**cp.swift structure**:
- Use `var grid: [[Int]] = Array(repeating: Array(repeating: 0, count: 9), count: 9)` for puzzle
- Use `var candidates: [[Int]]` for bitsets (Int type, bits 1-9 represent digits)
- Implement `propagate(row: Int, col: Int, digit: Int)` to eliminate candidates
- Implement `solve() -> Bool` with backtracking recursion
- Use global `var iterations = 0` counter, increment BEFORE validity check
- Parse .matrix file from CommandLine.arguments
- Initialize candidates: for empty cells, set to 0x3FE (bits 1-9), for filled cells set to 0

**Bitwise operations**:
- Check if digit available: `(candidates[row][col] & (1 << digit)) != 0`
- Remove candidate: `candidates[row][col] &= ~(1 << digit)`
- Add candidate back: `candidates[row][col] |= (1 << digit)`
- Count candidates: use `(0..<10).filter { (candidates[row][col] & (1 << $0)) != 0 }.count`

**Critical requirements**:
- Swift's bitwise operators: & (AND), | (OR), ~ (NOT), << (left shift)
- Must propagate constraints to row, column, and 3x3 box
- Backtracking must restore candidate sets correctly
- Print output format: puzzle, then "Solved in Iterations=67"
- **Avoid**: Don't use Set<Int> for candidates (bitset is faster and matches reference). Don't use tuple returns (keep it simple like C).
