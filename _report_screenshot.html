Scoring Methodology

The Composite Score (Ψ) uses the geometric mean of four performance ratios, an industry-standard method used by SPEC and Geekbench benchmarks.

The Baseline: C

ΨC = 1.0 (Reference Point)

The Formula

Ψ = 4√(ρtime ⋅ ρmem ⋅ ρiter ⋅ ρcpu)

where ρx = Valuesolver / ValueC

Metrics Used

  • ρtime : Total execution time ratio
  • ρmem : Peak memory usage ratio
  • ρiter : Algorithm iterations ratio
  • ρcpu : CPU time (user+sys) ratio

Why Geometric Mean?

  • • Prevents one metric from dominating
  • • Penalizes imbalanced performance
  • • Industry standard (SPEC, Geekbench)

Interpretation

  • Ψ = 1.0 : Matches C baseline exactly
  • Ψ < 1.0 : Outperforms C (lower is better)
  • Ψ > 1.0 : Underperforms C
